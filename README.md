# 轮动策略回测系统 —— Streamlit 可视化版

基于 rotation-backtest 技能的可视化网页端，支持自定义股票池、排序指标、买卖条件、仓位管理等完整回测功能。

## 功能特性

- 🎯 **完全可配置**：自定义股票池、排序公式、买卖条件
- 📊 **可视化界面**：Streamlit 交互式操作，无需编程
- 📈 **丰富图表**：净值曲线、回撤分析、持仓统计
- 🔄 **预设策略**：内置多种经典轮动策略模板
- 💾 **数据导出**：交易日志 CSV 下载

## 支持的指标

MA, EMA, RSI, MACD, ATR, BOLL, KDJ, returns, BIAS, quality_score, RSRS, volatility 等

## 快速开始

1. 访问部署的 Streamlit 应用
2. 选择预设策略或自定义配置
3. 点击"运行回测"
4. 查看净值曲线和统计指标

## 部署

部署到 [Streamlit Cloud](https://streamlit.io/cloud)
